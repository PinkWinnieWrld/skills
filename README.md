# skills

qb-skills
A complete skill progression system for QBCore servers, featuring:

Stamina → longer sprint duration

Strength → increased melee damage (players only)

Driving → improved vehicle handling

Shooting → reduced recoil & spread

Gym membership system

Skill decay over time

Skill notifications (ox_lib)

F1 skill menu with progress bars

Gym training using ox_target

XP from running, driving, shooting, and gym exercises

Features
Skill System
Four skills: Stamina, Strength, Driving, Shooting

XP gained from real gameplay actions

Skills stored in QBCore metadata

Configurable XP rates

Gameplay Effects
Stamina increases sprint duration

Strength increases melee damage

Driving improves handling, traction, and torque

Shooting reduces recoil and bullet spread

Gym System
Buy a membership (7 days by default)

Required for using gym equipment

ox_target interaction zones

Skill Decay
Skills decay over time if not used

Fully configurable per skill

UI
F1 menu using ox_lib context

Progress bars for each skill

Toast notifications when skills increase

Dependencies
This resource requires:

QBCore Framework

ox_lib

ox_target

ox_inventory

oxmysql

Make sure all dependencies are installed and started before this resource.
